# govcl-template

govcl template repo

govcl repo: https://github.com/ying32/govcl

适配`Go 1.17`以上版本.
## How to use

### Go project

两种方法:  

1. 把`go.mod`文件里的包名改成自己的.(可能会出现未知错误)
2. 把`go.mod`删了,重新`go mod init 包名`.

### GUI

改个项目名行了.

### CI

把`.github\workflows\go-build.yml`里面的所有`govcl-template`替换成自己的项目名.

### 其他

别忘了把`.github\FUNDING.yml`文件删了.

其他随意.
